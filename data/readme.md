## Data Description

The original dataset for this project consists of multiple large CSV files containing
transactional sales, purchase, inventory, pricing, and vendor information.

Due to file size limitations, the raw data files are not included in this repository.

### Raw Data Files (Not Included)
- begin_inventory.csv
- end_inventory.csv
- purchase_prices.csv
- purchases.csv
- sales.csv
- vendor_invoice.csv

### Included Data File
- **vendor_sales_summary.csv**

The `vendor_sales_summary.csv` file is a cleaned, filtered, and consolidated dataset
generated after data ingestion, transformation, and feature engineering steps.
It serves as the primary analytical dataset used for exploratory analysis,
statistical testing, and Power BI dashboard creation.

All analysis and insights presented in this project are based on this processed dataset.
